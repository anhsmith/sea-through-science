# STS logo — vector assets

Generated by the `logoSTS` repo (`logoSTS2.qmd` + `lockup.qmd`) — **do not edit
these by hand**; change the parameters there and re-render.

| file | use |
|---|---|
| `logo-fill.svg` | mark alone, white/light backgrounds (no keyline) |
| `logo-fill-white.svg` | mark alone, white/light backgrounds, white keyline |
| `logo-fill-navy.svg` | mark alone, on navy `#071739` |
| `logo-wire-black.svg` / `logo-wire-white.svg` | wireframe variants |
| `lockup-horiz.svg` | logo + wordmark, white backgrounds |
| `lockup-horiz-white.svg` | logo + wordmark, dark backgrounds |
| `lockup-navy.svg` | self-contained navy banner |
| `lockup-square.svg` | stacked square lockup, white backgrounds |

Palette is background-dependent by design: white-background assets use
`#54B5CB` / `#027DBF`; navy-background assets use the original
`#99D9EA` / `#00A2E8`. Wordmark is IBM Plex Sans Medium, baked to outlines —
no font needed anywhere. Everything scales cleanly; the fills are tuned to
stay legible at 34px height.
